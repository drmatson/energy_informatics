ğŸ“Œ What is Pandas?#

Pandas is a Python library for data analysis and manipulation.

  • Built on top of NumPy.

  • Provides two main data structures:

    • Series â†’ one-dimensional labeled array (like a column).

    • DataFrame â†’ two-dimensional table (like Excel or SQL).

  • Makes it easy to clean, transform, join, and analyze data.

âš¡ Itâ€™s the go-to library for working with datasets in data science, machine learning, and energy analytics.

ğŸ”€ Joins and Merges#

  • Inner Join â†’ only rows with matching keys (default).

ğŸ” Filtering with .query()#

  • Column names are used directly (without df["col"]).

In Pandas string methods, .str.replace() can work in two modes:

  • regex=False â†’ the pattern is treated as plain text.

  • regex=True (default) â†’ the pattern is treated as a regular expression (regex).

We need regex when:

  • We want to match patterns instead of exact text.

  • Example: replace any sequence of spaces, hyphens, or underscores with a single space.

âš¡ More Powerful Regex Example#

Suppose we want to replace any non-letter character ([^A-Za-z]) with a space:

labels = pd.Series(["  SOLAR  ", "wind-farm", "Hydro_Plant", "Bio*Mass!"])
clean = (labels.str.strip()
               .str.replace(r"[^A-Za-z]", " ", regex=True)  # replace non-letters
               .str.split()                                # remove extra spaces
               .str.join(" ")                              # join words back
               .str.title())

print(clean)

0          Solar
1      Wind Farm
2    Hydro Plant
3       Bio Mass
dtype: object

ğŸ” Key Takeaway#

  • Without regex â†’ you must replace characters one by one ("-", "_", "*", etc.).

  • With regex â†’ you can define a pattern once (like â€œnon-lettersâ€) and clean in a single step.

ğŸ‘‰ Regex makes string cleaning much more powerful and flexible, especially in messy energy datasets (plant names, region codes, sensor IDs).
