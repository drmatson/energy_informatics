üî§ Types in Python#

In Python, data types tell us what kind of value we are working with. Python is a dynamically typed language, meaning you don‚Äôt need to declare types explicitly ‚Äî Python figures it out for you.

Common Python Data Types:#

  • int üî¢ ‚Äì Whole numbers (e.g., 5, -42, 1000)

  • float ‚ûó ‚Äì Decimal numbers (e.g., 3.14, -0.5, 2.0)

  • str üìù ‚Äì Text strings (e.g., "Solar", 'Energy')

  • bool ‚úÖ/‚ùå ‚Äì Boolean values: True or False

  • list üìã ‚Äì Ordered, changeable collection (e.g., [1, 2, 3])

  • tuple üîí ‚Äì Ordered, but unchangeable collection (e.g., (1, 2, 3))

  • dict üìñ ‚Äì Key-value pairs (e.g., {"building": "1", "demand": 25.0})

  • set üß© ‚Äì Unordered collection of unique items (e.g., {1, 2, 3})

‚úÖ Rules (must follow)#

  • Must start with a letter or underscore (not a number)

  • Case-sensitive (Name and name are different)

üö´ Invalid examples:

  • Use UPPERCASE for constants:

    PI = 3.14159

  • Choose meaningful names (avoid single letters unless for math loops):

all_sources = {"solar", "wind", "hydro", "nuclear", "coal"}

print(all_sources - renewables)   # difference -> {'coal', 'nuclear'}
print(all_sources & renewables)   # intersection -> {'solar', 'wind', 'hydro'}
print(all_sources | renewables)   # union -> {'coal', 'nuclear', 'solar', 'wind', 'hydro'}

{'nuclear', 'coal'}
{'wind', 'hydro', 'solar'}
{'wind', 'hydro', 'coal', 'nuclear', 'solar'}

üîÅ Loops in Python#

Loops let you repeat actions without writing the same code multiple times. There are two main types:

  • for loop ‚Üí iterate over a sequence (list, tuple, dict, set, string).

  • while loop ‚Üí keep running while a condition is True.

üëâ Use for loops for going through collections üìã and while loops for conditions üîé.

‚ñ∂Ô∏è for Loop#

Used when you know how many times you want to repeat something (like going through a list of energy sources).

energy_sources = ["solar", "wind", "hydro"]

for source in energy_sources:
    print(f"Energy source: {source}")

Energy source: solar
Energy source: wind
Energy source: hydro

üìù List Comprehensions#

A list comprehension is a shorter way to create a new list by applying an operation to each item in an existing list.

üëâ List comprehensions are compact and efficient, often replacing loops when transforming data.

capacities = [100, 200, 300]
new_capacities = [c * 1.1 for c in capacities]  # increase each by 10%
print(new_capacities)
# [110.0, 220.00000000000003, 330.0]

[110.00000000000001, 220.00000000000003, 330.0]

energy_plants = [
    {"name": "SolarOne", "renewable": True},
    {"name": "WindFarmX", "renewable": True},
    {"name": "CoalPlantA", "renewable": False},
]

renewable_plants = [plant["name"] for plant in energy_plants if plant["renewable"]]
print(renewable_plants)
# ['SolarOne', 'WindFarmX']

['SolarOne', 'WindFarmX']

üîÑ while Loop#

Used when you don‚Äôt know exactly how many times the loop should run, but you have a condition to check.

capacity = 0

while capacity < 100:
    capacity += 25
    print(f"Current capacity: {capacity} MW")

Current capacity: 25 MW
Current capacity: 50 MW
Current capacity: 75 MW
Current capacity: 100 MW

üîß Functions in Python#

A function is a reusable block of code that performs a task.

  • Defined with the keyword def.

  • Can take parameters (inputs) and return values (outputs).

üëâ Functions help you organize, reuse, and simplify code ‚Äî essential for energy analytics, simulations, and machine learning projects.

‚ú® Basic Function#

def energy():
    print("Welcome to the Energy World!")
    
energy()

Welcome to the Energy World!

‚ö° Function with Parameters#

def energy_output(source, capacity_mw):

üîÑ Function with Return Value#

def increase_capacity(capacity, percent):
    return capacity * (1 + percent/100)

üß© Function with Default Parameter#

def efficiency(plant, percent=90):
    return f"{plant} efficiency is {percent}%"

‚ö° Lambda, map(), and filter() in Python#

üîπ Lambda Functions#

A lambda function is a small, anonymous function defined in one line using the keyword lambda.

üëâ Use filter() to select items ‚úÖ, and map() to transform them üîÑ. Both work great with lambda functions for compact, powerful code.

square = lambda x: x**2
print(square(5))  
# 25

25

üîπ filter()#

filter(function, iterable) keeps only the elements that satisfy the condition.

capacities = [50, 120, 200, 75, 300]
large_plants = list(filter(lambda c: c > 100, capacities))
print(large_plants)
# [120, 200, 300]

[120, 200, 300]

üîπ map()#

map(function, iterable) applies a function to each item of a list.

üìù Basic Example#

capacities = [100, 200, 300]
new_capacities = list(map(lambda c: c * 1.1, capacities))  # increase by 10%
print(new_capacities)
# [110.00000000000001, 220.00000000000003, 330.0]

[110.00000000000001, 220.00000000000003, 330.0]

üîπ reduce()#

The reduce() function (from the functools module) applies a function cumulatively to the items of a list, reducing it to a single value.

üëâ Use reduce() when you want to aggregate values into one result ‚Äî like calculating total energy capacity, total cost, or total emissions.

from functools import reduce

capacities = [100, 200, 300, 400]
total_capacity = reduce(lambda a, b: a + b, capacities)
print(total_capacity)
# 1000

1000
